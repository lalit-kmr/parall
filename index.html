How to Use

  1. Turn on your mobile Hotspot (or connect to one)
  2. Start the file server in the app
  3. That's it! You're ready to share

Select Files to Share:

  1. Choose specific files or
  2. Enable "Device Storage" to access all files (internal/external)

On the Receiver's Device:

  1. Open any web browser
  2. Enter the IP address shown on your phone: http://IP:PORT (Not HTTPS!)
  3. Don't forget the colon ":" between IP and PORT
